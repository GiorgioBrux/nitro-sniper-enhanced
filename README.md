## ✨ Features

## 🚀 Installation Methods

### Local Installation
```bash
# Clone repository
git clone https://github.com/giorgiobrux/nitro-sniper-enhanced nitro-sniper

# Install dependencies
cd nitro-sniper
npm install

# Configure and run
# 1. Edit .env file
# 2. Start application
node .
```

### Docker (Recommended)
Available on [DockerHub](https://hub.docker.com/r/giorgiobrux/nitro-sniper)
